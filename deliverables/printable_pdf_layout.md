# Printable PDF Layout — AfterClass Circuit Module

## Page Setup
- Paper: US Letter (8.5" × 11")
- Orientation: Landscape
- Margins: 0.5" on all sides
- Output: PDF for print

## Card Size Options
**Option A (Index Card):** 3" × 5"
- Best for readability and writing room.

**Option B (Trading Card):** 2.5" × 3.5"
- Best for portability and denser decks.

## Recommended Grid Layout
**For 3" × 5" cards:**
- 2 rows × 2 columns per page (4 cards per page)
- Gutter between cards: 0.25"

**For 2.5" × 3.5" cards:**
- 3 rows × 3 columns per page (9 cards per page)
- Gutter between cards: 0.2"

## Cut Guides
- Add light gray cut lines around each card boundary.
- Include crop marks if exporting from design software.

## Print Material
- Review copy: standard printer paper (20–24 lb)
- Student deck: matte cardstock, 350–400 gsm (14–16 pt)

## Export Notes
- Export as PDF, print-ready
- Embed fonts or convert text to outlines before export
- Use 300 dpi for crisp text and diagrams
